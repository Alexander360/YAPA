Welcome to YAPA Page.

YAPA is minimalistic desktop timer app for Pomodoro Technique users. For more info about Pomodoro Technique go to Pomodoro page or Wikipedia

Futures

These are the current features:

  • Configurable periods
  • Sound control
  • Pomodoro™ counter :)
  • Shows period progress on taskbar
  • Control app using taskbar jumplist
  • Select opacity for timer
  • Select light or dark theme for timer
